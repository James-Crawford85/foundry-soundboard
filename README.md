FOUNDRY SOUNDBOARD



Put all sounds into the "Sounds" folder, you can create new folders to organize the files.



Inside the "Sounds" folder, run the "Sounds2Json.ps1" this script will parse the "Sounds" folder and create the "soundboard.json" file which is needed by the module. 



Every time to add new files you need to rerun the "Sounds2Json.ps1"

To install, within Foundry on the Add-On Module tab, click on Install Module

In the Manafest URL at the bottom paste the following URL: https://github.com/James-Crawford85/foundry-soundboard/releases/download/1.0/module.json

Click Install
